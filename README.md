ğŸ¯ Project Objectives

The system will provide a dynamic and user-friendly library platform that:

- Enables users to search for books by title, author, category, or keyword.
- Displays book availability status (Available, Borrowed, Reserved).
- Allows registered users to:
  - Borrow books
  - Reserve unavailable books
  - Return borrowed books
  - View borrowing history and due dates
- Supports secure authentication for students and staff.
- Provides librarians/admins with tools to:
  - Add, update, or remove books
  - Manage users and memberships
  - Track loans, returns, and overdue items
  - Generate basic library reports and statistics
- Includes a dashboard showing:
  - Number of borrowed books
  - Due dates and overdue alerts
  - Notifications or announcements
